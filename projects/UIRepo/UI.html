Cursor Menu

A custom interactive trailing nav that morphs into a floating menu. Designed for modular integration. On hover, the nav morphs smoothly into a menu panel.

The component reverts back to its original state when the user's cursor moves away, eliminating the need for manual closing.

Gamification

The Gamification module is a compact sidebar item that introduces interactive game-like elements. It rewards user behavior, encourages exploration, and builds a sense of progression through simple achievements.

Built around if/else logic, the demo tracks user actions and responds with feedback by offering points and badges, motivating the user to engage with the content. A small proof of concept with broad potential—from onboarding and tutorials to product tours.
